# AI-Based Anomaly Detection Using Isolation Forest

This project uses the Isolation Forest algorithm to detect abnormal student behavior in a smart classroom environment.

## Features
- Attention Score Analysis
- Eye Focus Monitoring
- Activity Score Analysis
- Attendance Monitoring
- Anomaly Detection

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Isolation Forest

## Output
The model classifies student records as:
- Normal
- Anomaly
